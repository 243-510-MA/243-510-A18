Module Fermier:
Le module fermier est un module portable où les données de la serres sont disponibles pour le fermier.
Lorsqu'allumé, le module se connecte au réseau et demande les informations de température et d'humidité
aux plaquettes capteur. Le bouton de droite change de capteur et le bouton de gauche revient au capteur
précédent.

Module capteur:
Le module capteur a deux versions. 
Une version contenant un capteur d'humidité de sol(qui fait en même temps la lecture de la température).
Une version contenant un capteur de température de l'air.
-Version capteur d'humidité de sol:
Cette carte contient une communication i2c entre le PIC18F46J50 et le capteur. La lecture se fait à intervalle
prédéterminé. Par la suite, il envoie sa lecture d'humidité et de température au module PAN via le réseau.
-Version capteur de température de l'air:
Cette carte contient une communication SPI entre le PIC18F46J50 et le capteur. La lecture se fait à intervalle
prédéterminé. Par la suite, il envoie sa lecture d'humidité et de température au module PAN via le réseau
**Prendre note que les PCB sont les même pour les deux, ils contiennent simplement des composants différents.

Module PAN:
Le Module pan est celui qui:
1. Garde en mémoire tous les informations dans sa EEPROM.
2. Gère la communication entre tout les modules (Voir une communication STAR avec MIWI dans http://wikitge.cmaisonneuve.profweb.ca/images/MiWi-Intro_v2.1.pdf (MERCI FRED :D))
3. Fait une communication avec un Arduino pour mettre ses données sur le NET.
4. Affiche ses données sur un LCD
